# Arduino-vol-0_projects
This is a collection of my ardruino sketches for begginers, covering sensor integration, displays and basic electronics logic. 
I developed this projects when it was year 2025 and I was 14 years old, It was that time where I finnaly have my first arduino kit.

This arduino sketches covers sensors, RFID, OLED/LCD display, servos, LEDs, potentiometer and many more. It is a perfect reference for beginners looking to understand how to experiment individual components with arduino.
